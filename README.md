# üé∂MediaSense - Steelseries OLED Media Player

Media Sense is a project that is made to display the currently playing music on your system on a Steelseries keyboard with OLED screen.
It supports Steelseries Keyboards with a **128x40 pixel displays** (tested on an Apex Pro)

## ‚ú®Features
- Displaying currently playing songs using winsdk
- Automatically updates the OLED with current track informations
- Automatic scrolling titles, progressbar, and elapsed time and duration.
- Easy to setup

## üß∞Prerequisites
- You'll need Python 3.8 or higher, which can be installed [here](https://www.python.org/downloads/).
- You'll also need the [Steelseries GG](https://steelseries.com/gg/engine) software.

> Make sure to install python from the official python website and not from the Microsoft Store as it may causes issues during installation or execution.

## üñ•Ô∏èInstallation
  1. Download the [latest version](github.com/ShyFoxYT/MediaSense/release/latest) of the program and extract the files.
  2. Run ``install.bat`` to start the installation.
